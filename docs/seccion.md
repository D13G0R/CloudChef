# Configuración del Proyecto

En esta sección se describe cómo configurar el entorno de desarrollo y producción.

## Variables de Entorno

Asegúrate de configurar las siguientes variables de entorno:
- `DEBUG`
- `DATABASE_URL`
- `SECRET_KEY`

## Configuración de Bases de Datos

Puedes usar SQLite para desarrollo y PostgreSQL para producción. Consulta la [documentación oficial](https://www.postgresql.org/) para más detalles.
